Roadmap to Proof of Concept MVP

Introduction

cThink (computational thinking) a capstone project at The Knowledge House, is a website that focuses on the abstraction piece of the computational thinking model. This platform allows students to utilize the {D.A.B.} Draw. Act. Build. pedagogy, by teaching technical coding concepts. Providing a pathway for beginner coders to retain complex coding concepts by building and associating them with personalized analogies.


The pathway for creating an analogy is a pedagogy model called {D.A.B.} short for DRAW. ACT. BUILD. {D.A.B.} works by incorporating a teaching model that taps into the process of how information is retained in long-term memory.


The incorporation of a teaching pedgogy like {D.A.B.} in the context of teaching and learning technical concepts to code newbies is a 3 step process that will reinforce technical concepts through associations that will support 3 learning styles visual auditory, and kinesthetic.


The brain stores long-term memories in the hippocampus, which needs context to store information. The context, in this case, is not lines of code. It is the mnemonic and rehearsal that will cause the memory to move from working memory into long-term memory. A change that starts in the brainâ€™s hippocampus.
